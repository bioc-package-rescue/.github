# Bioconductor Package Rescue (.github Organization Repository)

This repository contains organization-level configuration and profiles for the **[Bioconductor Package Rescue](https://github.com/bioc-package-rescue)** organization.

For the profile page displayed on the organization landing page, see the **[profile/README.md](profile/README.md)**.

---

## 🔍 What is this organization for?

When a Bioconductor package starts failing its daily builds (resulting in `ERROR` or `TIMEOUT` status) or is scheduled for deprecation, it risks being removed from the next Bioconductor release. 

Our mission is to **rescue** these packages by automatically identifying, diagnosing, and fixing build and check issues. We do this by combining **automated AI coding agents** (like Google Antigravity) with a centralized **GitHub Actions (GHA) testing infrastructure**.

For more details on how it works and guidelines for package maintainers, please visit our organization landing page or read the [profile README](profile/README.md).
